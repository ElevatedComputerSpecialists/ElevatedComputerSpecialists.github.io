PREVENT PROBLEMS

Identify and analyze server “events” for potential problems. Mac OS and Windows environments can have sporadic “issues” if they are improperly configured. For example, a Mac should use CIFS instead of SMB when accessing file shares and be joined to the domain. Another flaw is expecting to increase throughput by connecting both NICs of a Windows Server to a switch as there is further configuration required with NIC teaming on the server and LACP on the switch. Packet loss will occur if LACP is not configured.


PROTECT YOUR SYSTEM

Remediate processes or services that fail. Security should be handled via a layered approach. Buying the best anti-virus program will not fully protect your critical business assets. We recommend anti-virus and anti-malware software, firewall with DPI and DPI-SSL, gateway anti-virus, and a third party spam filter. Other recommendations include configuring the switches to prevent access to unknown computers, securing all network equipment and servers in a locked, well ventilated room, and configuring all VLANs to manage network traffic while logically separating this traffic to increase security.



STAY UP-TO-DATE

Ensure security patches, service packs and anti-virus definitions are up-to-date. Anti-virus is always a losing race because it’s only protecting you against known threats: this year’s cryptowall virus will have a different signature than last year’s, meaning there will be little guarantee you’ll be protected. Keeping up with the viruses and using the most current ‘definitions’ of known viruses keeps the company as safe as possible. To this end, we deploy security patches to cover known exploits of the operating systems and software, protecting your data and network from hackers. The more up-to-date your system is, the more secure and protected it will be.


GET COMPLETE BACKUPS

Ensure that data backups are completed. Imagine sitting down to do ‘your job’ and the file you’ve spent 6-months building is gone. POOF! Imagine if everyone in the company experienced the same thing. How much lost productivity can your company sustain? It’s industry best practice for backups to be monitored by a tech — and periodicly tested in a restoration process. The best backup systems perform incremental backups, performing as often as once each hour. This keeps the company ready to recover from the most recent backup point. We offer a business continuity service that takes and stores regular backups every hour/day/week/month. A virtual copy of the server can be created on the device for employees to continue working from. In addition, we also provide a cloud backup option which can protect against flooding or fire damage.
